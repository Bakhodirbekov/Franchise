```css
/* Tailwind CSS Breakpoints */
default:  < 640px   (Mobile)
sm:       640px+    (Mobile Large)
md:       768px+    (Tablet)
lg:       1024px+   (Desktop)
xl:       1280px+   (Large Desktop)
2xl:      1536px+   (Extra Large)
```

**Используемые паттерны:**
